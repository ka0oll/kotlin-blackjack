카드
- [X] 카드는 총 52장이다.
    - [X] 2-9, ace(1, 11), king(10), queen(10), jack (10)

- [X] 카든 무늬
    - 하트, 스페이드, 클로버, 다이아몬드
    
블랙잭 게임
- [x] 참가자를 모집
- [x] 처음 카드를 2장을 준다.
- [x] 카드 1장을 준다.(hit)
  - [X] 카드가 21점 미만 일때만 준다. 

참가자

- [X] 받은 카드
- [X] 카드의 점수 계산.
