# kotlin-blackjack

카드
- [ ] 카드는 총 53장이다.
    - [ ] 2-9, ace(1, 11), king(10), queen(10), jack (10)

- [ ] 카든 무늬
    - 하트, 스페이드, 클로버, 다이아몬드
    
블랙잭 게임
- [ ] 참가자를 모집
- [ ] 처음 카드를 2장을 준다.
- [ ] 카드 1장을 준다.(hit)
- [ ] 현재 턴인 유저에게 카드를 받을지 물어 본다.

참가자
- [ ] 이름
- [ ] 받은 카드
- [ ] 카드 점수 계산. 
- [ ] 카드를 받을수 있는지 체크
